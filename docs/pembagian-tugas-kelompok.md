# Pembagian Tugas Kelompok

Gunakan format ini pada slide presentasi dan laporan.

| No. | Materi (Gabungan)                                             | Dikerjakan oleh | Catatan                                              |
| --- | ------------------------------------------------------------- | --------------- | ---------------------------------------------------- |
| 1   | Analisis kebutuhan + studi literatur + desain BPMN            | Anggota 1       | Referensi OWASP, NIST, alur proses bisnis            |
| 2   | Desain database MySQL + ERD + implementasi model/migration    | Anggota 2       | users, items, tracking_events, transaction_logs      |
| 3   | Implementasi auth + RBAC + input item + update status         | Anggota 3       | Register/login, role admin/supplier/kurir            |
| 4   | Implementasi enkripsi + hash chain + integrity checker        | Anggota 4       | AES/RSA, append hash, verifikasi tampering           |
| 5   | Audit trail + security testing + laporan PDF + PPT presentasi | Anggota 5       | Simulasi attack, ringkasan hasil, finalisasi dokumen |

Catatan:

1. Ganti Anggota 1-5 dengan nama asli anggota kelompok.
2. Jika ada anggota yang lebih kuat di coding, bisa pegang materi nomor 3 dan 4 bersama.
3. Pastikan setiap materi punya output terukur: dokumen, kode, dan bukti pengujian.
